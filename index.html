mirror

Repurposed as an I2P source mirror. For binary mirrors check:

  • https://github.com/eyedeekay/apt - Debian and Ubuntu Packages
    • eyedeekay.github.io/apt
  • https://github.com/eyedeekay/files - Windows, Mac OSX, IzPack and NSIS releases
    • eyedeekay.github.io/files
  • https://github.com/eyedeekay/fdroid - Android releases
    • eyedeekay.github.io/fdroid

This repository is different in that it pulls in source code but it does not treat them as submodules and does not check them into the repository. Instead it just pulls in all the updates to all the branches, all the tags, and drops it in a directory on a web server, .git and all.echelon.i2p

Using this tool requires git-extras .
